# 台灣大樂透新世代鐵律預測系統

本系統為全新獨立運算核心，只使用台灣彩券官方歷史開獎資料，不沿用任何舊預測函式。

## 運算與驗證

- 2,152期正式歷史資料，起始2007-01-02，最新2026-07-14。
- 主號與特別號分開建立模型、回測及排名。
- Bayesian平滑、四段EWMA、實證遺漏風險、條件共現與趨勢狀態模型。
- 每期只能使用之前資料，以520期時間序列走步驗證。
- 模型權重只依前120期表現決定，單一模型不得超過30%。
- 機率向公平開獎先驗收縮80%，避免製造虛假高信心。
- 發布門檻未通過時，禁止覆蓋最後一份有效手機戰報。

目前520期驗證：主號前12碼平均命中1.5096，隨機基準1.4694；特別號前三碼平均命中0.0712，隨機基準0.0612。這是歷史驗證結果，不代表未來保證。

## 戰報規格

核心決策、逐號驗算、短包強牌、低機率獨立區、上期誤開檢討、戰果驗算、模型回測、每月總結、鐵律守門與預測生命週期。

## 執行

```powershell
python update.py
python verify.py
```

GitHub Actions每3小時檢查一次；週二、週五21:25至22:30每5分鐘加密輪詢。只有官方資料驗證、520期能力門檻與全部自測通過後才會同步手機雲端。

獨立手機雲端版位於 `mobile_cloud`，Pages專屬入口為 `/mobile_cloud/`，並與電腦戰報逐檔同步驗證。

## 不可變動的自動更新鐵律

`IRONLAW.json` 鎖定官方抓號、資料驗證、上期結算、全量重算、五處同步、失敗禁止發布與保留最後有效戰報。發布流程會逐次檢查這些規則，任何一條缺少即中止。
